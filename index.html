¡Hola!

Bienvenid@ a mi portafolio web personal donde podrás saber un poco más sobre mí y los proyectos en los que eh trabajado.

Ser un desarrollador web Full stack significa que convina ambas especialidades: Font-end y Back-end.

La programación web Front-end prioriza la éstetica visual de la página web tales como la paleta de colores, imágenes, fuente de la letra, animaciones, etc. De tal manera que el usuario que esté visualizando la página se sienta cómodo y le agrade lo que está viendo. Los lenguajes bases son: HTML, CSS, JavaScript y React.

El desarrollador Back-end se especializa más en como es el sitio "por detrás" se encarga más de alguna base de datos que pueda requerir el sitio, si quiere traer información de algún servidor, etc. Los lenguajes más usados son: PHP, MySQL, Python, JavaScript.
